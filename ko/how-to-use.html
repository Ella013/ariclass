아리클래스 사용 방법

아리클래스는 몇 초 만에 맞춤 교육용 학습지를 만들 수 있는 서비스입니다. 이 가이드를 따라 학습지 생성기를 시작해보세요.

시작하기

아리클래스로 학습지를 만드는 것은 간단하고 회원가입이 필요 없습니다. 다음 기본 단계를 따라해보세요:

  1. 학습지 유형 선택: 홈 페이지에서 단어 찾기, 문장 배열, 십자말풀이, 단어 연결, 빙고 게임 중 원하는 것을 선택하세요.
  2. 내용 입력: 단어, 문장, 어휘 등을 입력 칸에 입력하거나 붙여넣기 하세요.
  3. 옵션 설정: 레이아웃 형식, 간격, 글씨 크기 등 다양한 옵션을 조절하세요.
  4. 생성 & 미리보기: "학습지 생성" 버튼을 클릭해 학습지를 만들고 오른쪽에서 미리보기를 확인하세요.
  5. 인쇄: 인쇄 버튼을 눌러 브라우저에서 바로 출력하세요.

학습지 유형

단어 찾기

맞춤 어휘로 단어 찾기 퍼즐을 만들어보세요. 어휘 연습과 철자 익히기에 최적입니다.

  • 입력 칸에 단어를 입력하세요
  • 대문자 또는 소문자 선택 가능
  • 격자 크기 선택 (8x8, 11x11, 15x15)
  • 대각선 단어, 뒤집힌 단어 등 옵션 설정 가능

문장 배열

학생들이 단어 순서와 문장 구조를 익힐 수 있도록 문장 배열 학습지를 만들어보세요.

  • 마침표, 물음표, 느낌표로 끝나는 문장을 한 줄씩 입력하세요
  • 1단 또는 2단 레이아웃 선택 가능
  • 문항 간격 조절 (좁게, 보통, 넓게)
  • 섞인 문장의 첫 단어 대문자 표시 옵션

십자말풀이

어휘 연습과 개념 복습을 위한 십자말풀이 퍼즐을 만들어보세요.

  • 단어와 힌트를 입력하세요
  • 십자말풀이 격자가 자동으로 생성됩니다
  • 채점에 편리한 정답지가 포함됩니다

단어 연결

관련 단어, 개념, 정의를 연결하는 학습지를 만들어보세요.

  • 관련 항목 쌍을 입력하세요 (단어, 정의, 번역 등)
  • 자동으로 섞어서 번호를 매겨줍니다
  • 번호가 매겨진 정답으로 "정답 보기" 기능 포함

빙고 게임

어휘 연습과 복습 활동을 위한 맞춤 빙고 게임을 만들어보세요.

  • 어휘 단어를 입력하세요
  • 보드 크기 선택 (3x3, 4x4, 5x5)
  • 플레이어 수 선택 (각각 다른 보드 생성)
  • 글씨 크기 조절 (작게, 보통, 크게)
  • 학습지 제목 설정 가능

선생님을 위한 팁

  • 내용 미리 준비: 시작 전 어휘, 문장, 개념을 미리 준비하세요. 시간을 절약하고 정확도를 높일 수 있어요.
  • 현재 수업 활용: 현재 교육과정에 맞는 학습지를 만들어 학습을 강화하고 관련 연습을 제공하세요.
  • 난이도 조절: 격자 크기, 간격, 글씨 크기 등 다양한 설정을 사용해 학생 수준에 맞추세요.
  • 나중을 위해 저장: 같은 학습지를 여러 반을 위해 여러 장 출력하거나, 내용을 저장해 나중에 다시 만들 수 있어요.
  • 활동 조합: 같은 어휘로 다양한 유형의 학습지를 만들어 다채로운 연습 기회를 제공하세요.

학습지 인쇄하기

모든 학습지는 A4 용지 크기에 최적화되어 있습니다. 인쇄 시:

  • 미리보기 영역의 "학습지 인쇄" 버튼을 클릭하세요
  • 인쇄 대화상자에서 용지 크기가 "A4"로 설정되어 있는지 확인하세요
  • 여백은 기본값(보통 20mm)으로 설정하세요
  • 빙고처럼 여러 페이지가 있는 경우, 모든 페이지가 자동으로 인쇄됩니다
  • 메뉴와 입력 영역은 인쇄 시 자동으로 숨겨집니다

자주 묻는 질문

회원가입이 필요한가요?

아니요, 아리클래스는 완전히 무료이며 회원가입이 필요 없습니다. 웹사이트에 접속해 바로 학습지를 만들 수 있어요.

상업적으로 사용할 수 있나요?

아리클래스로 만든 학습지는 개인 및 교육 목적으로 사용하기 위한 것입니다. 상업적 이용은 이용약관을 확인해주세요.

입력한 내용이 서버에 저장되나요?

아니요, 입력한 모든 내용은 브라우저에서 로컬로 처리됩니다. 학습지 내용을 서버에 저장하거나 전송하지 않습니다.

어떤 과목의 학습지를 만들 수 있나요?

주로 영어와 수학을 위해 설계되었지만, 어휘, 연결, 퍼즐 기반 활동을 사용하는 어떤 과목에도 활용할 수 있습니다.

지금 시작할 준비가 됐나요?

홈 페이지를 방문해 모든 학습지 유형을 둘러보고 오늘 첫 번째 학습지를 만들어보세요!
