아리클래스 사용 방법

아리클래스는 몇 초 만에 맞춤형 교육 학습지를 만들 수 있는 서비스입니다. 아래 가이드를 따라 시작해보세요.

시작하기

아리클래스로 학습지 만들기는 간단하며 회원가입이 필요 없습니다. 다음 단계를 따라해 보세요:

  1. 학습지 유형 선택: 홈페이지에서 단어 찾기, 문장 배열, 십자말풀이, 연결하기, 빙고 게임 등을 선택하세요.
  2. 내용 입력: 단어, 문장, 어휘를 입력 칸에 직접 입력하거나 붙여넣으세요.
  3. 설정 조정: 레이아웃, 간격, 글자 크기 등 원하는 옵션을 설정하세요.
  4. 생성 및 미리보기: '학습지 생성' 버튼을 클릭하면 오른쪽에서 미리볼 수 있습니다.
  5. 출력: 출력 버튼을 눌러 브라우저에서 바로 인쇄하세요.

학습지 종류

단어 찾기

맞춤 어휘로 단어 찾기 퍼즐을 만들어보세요. 어휘 연습과 철자 학습에 좋습니다.

  • 단어 최대 10개까지 입력
  • 대문자 또는 소문자 선택
  • 그리드 크기 선택 (8×8, 11×11, 15×15)
  • 대각선, 거꾸로 쓰기 등 옵션 설정

문장 배열

문장을 섞어서 어순과 문장 구조를 연습할 수 있는 문제를 만드세요.

  • 마침표, 물음표, 느낌표로 끝나는 문장 입력
  • 1열 또는 2열 레이아웃 선택
  • 간격 조절 (좁게, 보통, 넓게)
  • 섞인 문장의 첫 글자 대문자 옵션

십자말풀이

어휘 연습과 복습을 위한 십자말풀이를 만들어보세요.

  • 단어와 힌트 입력
  • 자동으로 십자말 그리드 생성
  • 채점용 정답지 포함

연결하기

관련 단어, 개념, 뜻을 연결하는 문제를 만드세요.

  • 관련 항목 쌍 입력 (단어, 뜻, 번역 등)
  • 자동으로 섞어서 번호 표시
  • 번호가 있는 정답 확인 기능

빙고 게임

다양한 크기의 맞춤형 빙고 카드를 만들어보세요.

  • 어휘 단어 입력
  • 보드 크기 선택 (3×3, 4×4, 5×5)
  • 플레이어 수 선택 (각자 고유한 카드)
  • 글자 크기 조절 및 학습지 제목 설정

선생님을 위한 팁

  • 내용 미리 준비: 시작 전에 어휘, 문장, 개념을 미리 정리해두세요.
  • 현재 수업 활용: 현재 커리큘럼에 맞는 학습지를 만들어 학습을 강화하세요.
  • 난이도 조절: 그리드 크기, 간격, 글자 크기를 조절해 학생 수준에 맞추세요.
  • 나중을 위해 저장: 같은 학습지를 여러 학급에 출력하거나 내용을 저장해두세요.
  • 다양한 활동 조합: 같은 어휘로 다양한 유형의 학습지를 만들어 다채로운 연습 기회를 제공하세요.

학습지 출력하기

모든 학습지는 A4 용지에 최적화되어 있습니다. 출력 시:

  • 미리보기 영역에서 '학습지 출력' 버튼 클릭
  • 인쇄 설정에서 용지 크기를 'A4'로 선택
  • 여백은 기본값(20mm)으로 설정
  • 빙고처럼 여러 플레이어 카드가 있는 경우 모두 자동으로 출력
  • 출력 시 입력 영역과 메뉴는 자동으로 숨겨짐

자주 묻는 질문

회원가입이 필요한가요?

아니요. 아리클래스는 완전 무료이며 회원가입 없이 바로 사용할 수 있습니다.

이 학습지를 상업적으로 사용할 수 있나요?

학습지는 개인 및 교육 목적으로 사용하도록 설계되었습니다. 상업적 사용은 이용약관을 확인해 주세요.

입력한 내용이 서버에 저장되나요?

아니요. 입력하신 내용은 브라우저에서 로컬로 처리되며, 서버에 저장하거나 전송하지 않습니다.

어떤 과목의 학습지를 만들 수 있나요?

주로 영어와 수학을 위해 설계되었지만, 어휘, 매칭, 퍼즐 활동을 활용하는 모든 과목에 적용할 수 있습니다.

지금 시작해볼까요?

홈페이지로 이동해서 모든 학습지 유형을 살펴보고 첫 번째 학습지를 만들어보세요!
