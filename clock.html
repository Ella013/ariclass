How to Create Your Telling Time Worksheet

Simple Steps to Create

  1. Choose a mode: read the time or draw the hands
  2. Select the time interval (every hour, 30, 15, or 5 min)
  3. Set the number of clocks per page
  4. Click Generate for a new set of times
  5. Print your worksheet

Educational Benefits

  • Teaches analog clock reading skills
  • Builds number sense and time concepts
  • Two modes for reading and writing practice
  • Adjustable difficulty with time intervals

Tips for Teachers

  • Start with "every hour" for beginners
  • Use "draw the hands" for advanced practice
  • Mix 6 and 9 clock layouts for variety
  • Great for ESL/EFL time vocabulary lessons
