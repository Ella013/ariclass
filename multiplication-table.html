How to Create Your Multiplication & Division Worksheet

Simple Steps to Create

  1. Enter a title for your worksheet
  2. Choose operation type (Multiplication, Division, or Mixed)
  3. Select difficulty level (Easy, Medium, or Hard)
  4. Set the number of pages
  5. Click "Generate Worksheet"
  6. Print your worksheet using the print button

Difficulty Levels

  • Level 1: 1-digit × 1-digit (up to 9×9)
  • Level 2: 2-digit × 1-digit (e.g. 47×6)
  • Level 3: 2-digit × 2-digit (e.g. 34×28)
  • Division problems always have whole number answers

Tips for Teachers

  • Start with Level 1 for beginners
  • Use Mixed to practice both operations
  • Print multiple pages for timed drills
  • Use "Show Answers" to create an answer key
